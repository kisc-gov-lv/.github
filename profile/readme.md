# Kultūras informācijas sistēmu centrs (KISC) — GitHub organizācija

## Par KISC

**Kultūras informācijas sistēmu centrs (KISC)** ir valsts iestāde, kas darbojas **Kultūras ministrijas** pakļautībā ar mērķi nodrošināt **informācijas un komunikācijas tehnoloģiju (IKT) pārvaldību, drošību un pieejamību kultūras sektorā**. :contentReference[oaicite:1]{index=1}

KISC atbalsta kultūras mantojuma saglabāšanu, nodrošinot digitālās infrastruktūras un risinājumu attīstību bibliotēkām, arhīviem, muzejiem un citām institūcijām, kā arī īsteno tehnoloģiju projektus, lai veicinātu informācijas pieejamību sabiedrībai. :contentReference[oaicite:2]{index=2}

---

## Šīs organizācijas mērķis

Šī GitHub organizācija ir paredzēta **tehnisko resursu, referenču un atvērtā koda materiālu** publicēšanai, kas saistīti ar:

- IKT risinājumiem kultūras un atmiņas institūcijām
- Drošības un pieejamības uzlabošanu digitālajās platformās
- Automatizācijas un modernu tehnoloģiju (tai skaitā AI) integrāciju
- Standarta pieejām, arhitektūrām un praksēm atkārtojamam ieviešanai
- Projektiem, demonstrācijām un tehniskiem piemēriem

KISC nodrošina, ka **valsts kultūras nozares digitālā infrastruktūra** ir droša, pieejama un ilgtspējīga. Modernu IKT risinājumu izmantošana palīdz:
- saglabāt un digitalizēt kultūras mantojumu,
- padarīt to pieejamu plašākai sabiedrībai,
- stiprināt IKT drošību un ilgtspēju.

# Culture Information Systems Centre (KISC) — GitHub Organization

## About KISC

The **Culture Information Systems Centre (KISC)** is a state institution under the **Ministry of Culture** of Latvia, tasked with ensuring **centralized management, security, and availability of information and communication technologies (ICT)** within the cultural sector. :contentReference[oaicite:3]{index=3}

KISC supports the preservation of cultural heritage by developing and maintaining digital infrastructure solutions for libraries, archives, museums, and other cultural and educational institutions, and by implementing projects that make information more accessible to the public. :contentReference[oaicite:4]{index=4}

---

## Purpose of this GitHub Organization

This GitHub organization is intended to host **technical resources, references, and open source content** related to:

- ICT solutions for culture and memory institutions
- Accessibility and security improvements for digital platforms
- Automation tools and integration of modern technologies (including AI)
- Standard approaches, architectures, and reproducible practices
- Projects, demos, and technical examples

## Why This Matters

By driving secure, accessible, and sustainable digital infrastructure in the cultural sector, KISC helps:
- preserve and digitize cultural heritage,
- broaden public access to cultural content,
- strengthen ICT security and resilience.
